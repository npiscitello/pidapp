# What am I?

This application is a GUI demonstration of a PID controller. Right now I am implementing the GUI -
PID functions to come soon!

# Compile

To make both debug and release versions:

    make
    
or

    make all

debug:

    make debug

release:

    make app

To start fresh:

    make clean
